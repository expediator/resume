# Akshansh Yadav — Resume (Windows-OS style)

🔗 **Live:** https://expediator.github.io/akshansh-resume/

An interactive resume styled as a Windows desktop — clickable, draggable popup
windows with real `–  ▢  ✕` window controls, a taskbar with pinned apps and a
live clock, and a music player that autoplays soft R&B on open.

## Features
- Desktop icons + taskbar open draggable, closable "windows" for Profile,
  Education, Skills, Projects, Experience, Achievements, Interests & Contact.
- Auto-playing music player (play/pause, skip, mute) with visible song credits.
- **Music Library** window: an embedded Spotify R&B playlist plus a YouTube
  shortlist you can pick from.
- No build step — plain HTML/CSS/JS.
